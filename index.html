About Me

I'm an mechanical engineer by training who added programming to his skillset. The future is moving towards physical products that have smarts and both of these backgrounds gives me a better perspective in developing and leading these new products to the field.


As more and more products start to leverage the IoT concept, the worlds of computer and mechanical are becoming intertwined and this opens up a vast arena to use this new combination to improve lives around the globe. With computers, the limit of what can be accomplished is only limited to the ideas that are generated. By adding a mechanical component we can bring that limitless potential to the real world.

My Projects


Open World Platformer

This was the first game I made. I wanted to make a tribute to the game that started it for many people, Super Mario World, and I wanted to test my skills in trying to make a similar game. The enemies and environment is generated randomly and the speed of the game increases as the player scores more points.


Eaze Calendar

This was my first full-stack app. It was made using the MEN stack. Users can signup and can add multiple calendars to track their lives. You can add unlimited calendars for school, work, personal, etc. Users can add events to specific calendars to have them appear on their main calendar. Users can also add tasks which are less formal and serve as reminders of things that need to be accomplished.


findGrind

This is a de-coupled full stack app to help connect people who have one-off jobs and people for looking for some extra cash. The front end is made using React and the backend API is made using Node.js and Express.js. The app incorporates geolocating and Google Maps to easily match users with local jobs. The app is hosted on Heroku.


doStuf

This app makes it easy for people to find events in their city. It pulls events from various sources and allow users to track them in a single place. Users can customize the categories they see and can save individual events to their queue. The front end is created using React and the back end is made with Django using a PostgreSQL database.


SafeT

SafeT is an app to help people quickly and easily compare crash test ratings for different vehicles. It pulls data from the IIHS and the NHTSA and allows the user to compare up to 3 cars at the same time. It also pulls in recall information for each car to help the user make the most well informed decision. The front end is made with React and the backend utilized Django with a PostgreSQL database.
